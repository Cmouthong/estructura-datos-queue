Un **Planificador de CPU** que gestiona la ejecución de tareas mediante una cola con prioridades, y un **Sistema de Cola Bancaria** que maneja la atención secuencial de clientes mediante una cola FIFO (First In, First Out). Ambos proyectos ilustran conceptos fundamentales de estructuras de datos, implementados en Java con un enfoque orientado a objetos, proporcionando ejemplos prácticos de uso en situaciones del mundo real.
